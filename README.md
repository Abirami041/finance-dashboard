# Finance Dashboard (R Shiny)

This project is an interactive dashboard developed using R Shiny to manage and analyze personal financial data. It allows users to track income, expenses, commitments, and savings in a structured and visual manner.

## Overview

The application provides a simple interface for recording financial activities and generates real-time insights to support better financial decisions.

## Key Features

* Track income and categorize expenses
* Manage financial commitments (expenses and savings)
* Calculate savings and remaining balance dynamically
* Visualize spending patterns using charts
* Display basic financial insights based on user data

## Technologies Used

* R
* Shiny
* ggplot2
* dplyr

## Running the Application

To run the application locally, open the project in RStudio and execute:

```r
shiny::runApp()
```

## Notes

This project is designed as a lightweight financial tracking tool and can be extended with features such as persistent data storage, authentication, and advanced analytics.
